# Simple-Frontend-Backend-Calculator
Simple calculator made with FastApi and uvicorn backend and js/css

## Pre-install
* Install project
* cd into project directory
* install requirements into your global python or venv: pip install -r requirements.txt
* run backend server in the first terminal: uvicorn main:app --reload
* run any server that would hold frontend, I used python http server: python -m http.server 8001
* open localhost:8001/calculator.html
